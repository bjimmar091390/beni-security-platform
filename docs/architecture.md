# BENI Architecture

BENI is an AI-driven cybersecurity system designed to:

- Read security logs
- Analyze user behavior
- Assign risk scores
- Detect suspicious activity patterns

## Current Modules

### 1. Log Reader
Reads raw CSV security logs.

### 2. Risk Engine
Assigns risk scores based on user activity.

## Future Modules

- Pattern detection engine
- Real-time anomaly detection
- AI explanation layer
